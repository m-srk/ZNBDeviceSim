# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-22

### Added
- Initial release of ZNBDeviceSim
- Zoned Null Block device simulator with sequential write enforcement
- Rock-based interface for data object management
- Zone reset functionality
- 4KB page-level allocation and tracking
- Comprehensive unit tests
- Documentation and examples
- MIT License

### Features
- Support for multiple zones with configurable sizes
- Sequential write pointer per zone
- Page allocation on-demand
- Rock tracking via linked lists
- Detailed debug logging
